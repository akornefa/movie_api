Documentation Page

To build the server-side component of a “movies” web application. The web application will provide users with access to information about different movies, directors, and genres. Users will be able to sign up, update their personal information, and create a list of their favorite movies.

Business Logic URL HTTP Method Request body data format Response body data format
Return a list of ALL movies to the user /movies GET none A JSON object holding data about all movies
Return data about a single movie by title to the user /movies/[title] GET none A JSON object holding data (description, genre, director, image URL, whether it’s featured or not) about a single movie by title to the user.
Example: { description: "A 2000 romantic comedy-drama film based on the 1999 novel Chocolat by the English author, Joanne Harris." genre: "drama", director: "Lasse Hallstrom", image URL: "https://en.wikipedia.org/wiki/Chocolat_(2000_film)#/media/File:Chocolat_sheet.jpg" featured: "yes" }
Return data about a genre /movies/[genre] GET none A JSON object holding data about movies under the given genre.
Return data about a director /movies/director/[name] GET none A JSON object holding data (bio, birth year, death year) about a director.
Example: { name: "Lasse Hallstrom", bio: "Lars Sven "Lasse" Hallström is a Swedish film director", birth year: "1946 ", death year: "n/a" }
Register new user /users/register POST A JSON object holding data about a new user.
Example: { username: "aakyea", password: "sowej123!", birth year: "1990" } 		A JSON object holding data about the user who was just added, including an ID.
Example: { id: 3, email: "aakyea@gmail.com" username: "aakyea", password: "sowej123!", birth year: "1990" }
Allow users to update their user info (username) /users/username/[username] PUT none A text message indicating whether their information has been added
Allow users to add a movie to their list of favorites /users/title/[title] PUT none A text message indicating whether the movies was successfully added
Allow users to remove a movie from their list of favorites /users/title/[title] DELETE none A text message indicating whether the movies was successfully deleted
Allow existing users to deregister /users/email/[email] DELETE none A text message indicating whether the user's email has been successfully deregistered.
